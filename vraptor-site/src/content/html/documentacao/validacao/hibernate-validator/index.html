Hibernate validator

O VRaptor também suporta integração com o HibernateValidator. No exemplo anterior para validar o objeto Funcionario usando o Hibernate Validator basta adicionar uma linha de código: public adiciona(Funcionario funcionario) { validator.checking(new Validations(){{ that("erro","nomeNaoInformado", !funcionario.getNome().isEmpty()); //Validação do Funcionario com Hibernate Validator and(Hibernate.validate(funcionario)); }}) dao.adiciona(funcionario); }
